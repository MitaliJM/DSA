## Sorting Algorithm

A Sorting Algorithm is used to rearrange a given array or list elements according to a comparison operator on the elements.

### Applications

    * Content Searching
    * Editing
    * Text Processing
    * Signal Processing
    * Virus Scanning
    * Bioinformatics(DNA Strings)

### Algorithms

    * Selection Sort
    * Bubble Sort
    * Insertion Sort
    * Merge Sort
    * Quick Sort
    * Heap Sort
